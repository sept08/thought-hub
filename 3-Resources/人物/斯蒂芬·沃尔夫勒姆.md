## 人物简介

斯蒂芬·沃尔夫勒姆是计算科学领域最具影响力的人物之一。15岁开始发表物理学研究论文，20岁获得加州理工学院粒子物理学博士学位。他在24岁时创立了 Wolfram Research，开发了 Mathematica —— 至今仍是世界上最强大的计算系统之一。

他的核心思想体系围绕 **计算思维 (Computational Thinking)**：将一切知识转化为可计算的形式。从元胞自动机到 Wolfram Physics Project，他始终试图用简单的计算规则揭示宇宙的基本结构。

## 核心思想与方法论

### 1. 计算思维
- 用计算方法思考问题，将世界现象建模为可计算的过程
- 强调 **符号计算 (Symbolic Computation)** 与 **知识工程 (Knowledge Engineering)** 的结合
- 提出 **CAG (Computation-Augmented Generation)**：将 LLM 与精确计算结合，消除幻觉

### 2. 计算等价性原理 (Principle of Computational Equivalence)
- 即使极其简单的系统也能产生极其复杂的行为
- 自然界的复杂性与人工计算的复杂性在本质上等价
- 出自《A New Kind of Science》(2002)

### 3. Wolfram Physics Project
- 试图用简单的图重写规则推导出空间、时间和量子力学
- 将物理学的基础归结为计算过程
- 持续进行中：[wolframphysics.org](https://www.wolframphysics.org/)

### 4. AI 观点：计算增强生成
- LLM 擅长概率生成，但不擅长精确计算和事实验证
- Wolfram 技术为 LLM 提供 **可靠的计算基础层**
- 2026年2月推出 **Wolfram Foundation Tool**，通过 MCP 协议为 LLM 系统提供计算能力
