## 一、基本世界观（六条）

世界观是回答"世界如何运转"的底层假设和认知框架。你可以在很大程度上自主选择价值观和人生观，而对于世界观，却是你认识到什么程度，就得接受到什么程度——因为这个世界不是按照你的意志创造的。

### 第一条：叙事——宇宙的第一性原理

#### 1.1 第一性原理的追问

**定义**：探讨"这个宇宙或世界"的第一性原理是什么？

**思考与演变**：
- 早期认知：世界的第一性原理是物理定律，其他学科皆由其派生
- 后来的疑问：物理定律背后是否还有更深层的存在？
- 疑问来源：物理学界意识到，我们所处的宇宙可能并非唯一，可能存在无穷多各式各样的其他宇宙，那些宇宙未必遵循我们这个宇宙的物理定律

#### 1.2 沃尔夫勒姆的探索：计算比物理更基础

[[斯蒂芬·沃尔夫勒姆]]发起 Wolfram Physics Project (WPP)，目的是从更基本的原理出发推导出整个物理学。

沃尔夫勒姆认为：物理定律不是最底层的"第一性原理"，真正的第一性原理是**计算规则**。

##### 计算 ≠ 数学

这是沃尔夫勒姆思想体系的基石：

| | 数学 | 计算 |
|--|------|------|
| **研究对象** | 结构、数量、关系 | 过程、变换、演化 |
| **核心操作** | 证明、推导 | 执行、运行 |
| **典型工具** | 方程、定理、证明 | 规则、程序、模拟 |
| **思维方式** | "这个方程的解是什么？" | "按照这条规则运行下去会发生什么？" |
| **确定性来源** | 逻辑推导 | 规则执行 |

**例子**：
- 数学问题：给定方程 x² + y² = 1，求满足条件的点
- 计算问题：给定一条规则，从初始状态开始执行 1000 步，系统会变成什么样？

##### 什么是"计算规则"？

最简单的例子——**元胞自动机规则 110**：
```
规则：一行格子，每个格子是黑或白
      下一行的状态由上一行的3个格子决定
      
      比如：
      黑黑黑 → 白
      黑黑白 → 黑
      黑白黑 → 黑
      ...（共8种情况）
```

这**不是方程**，**不是公式**，就是一条**操作指令**："如果你看到这个图案，就把它变成那个"

Wolfram Physics Project 的核心是**图重写规则**：
```
初始状态：一个简单的图（点和边）
规则：找到匹配的子图 → 替换成另一个子图
不断执行 → 图变得越来越复杂 → 涌现出空间、时间、物质
```

##### 为什么计算比数学更基础？

- 数学擅长描述"状态"，不擅长描述"过程"
- 计算不需要方程，只需要规则
- 从简单规则可以涌现出无限复杂的行为
- **宇宙本身可能就是一个计算过程**——不断执行规则，从未停止

> 数学是人类发明来描述静态结构的工具。计算是自然界本身的运作方式。物理定律不是方程，而是计算规则执行后涌现出来的规律。

##### 为什么必须假设时空是离散的？

**核心问题：连续的东西"不可计算"**

| 连续 | 离散 |
|------|------|
| 一段线段包含**无穷多个点** | 一段线段包含**有限个格子** |
| 信息量**无穷大** | 信息量**有限** |
| 无法用有限存储表示 | 可以用有限存储表示 |
| 无法用有限步骤处理 | 可以用有限步骤处理 |

**计算的本质要求**：
- 计算需要存储状态 → 有限存储才能存储
- 计算需要执行步骤 → 有限步骤才能执行
- 计算需要输出结果 → 有限结果才能输出

**连续时空的困境**：如果空间连续，一个立方厘米的空间包含无穷多个点 → 要描述它的完整状态，需要无穷大的存储 → 这就**根本无法计算**

**沃尔夫勒姆的逻辑链条**：
```
前提：宇宙的第一性原理是计算规则
推论：宇宙必须是可计算的
再推论：宇宙必须是离散的（否则不可计算）
```

所以沃尔夫勒姆假设：
- **空间** = 一堆离散的点（他称为"atoms of space"，空间的原子）
- **时间** = 一拍拍的离散步骤

**目的**：把宇宙还原到最底层的可计算单元，从而探求是什么样的规则，通过不断计算，最终得到了我们这套物理定律的宇宙。

> 传统物理学用连续方程描述宇宙，是人类的**数学工具**，不是宇宙的**真实结构**。
> 
> 宇宙本身可能是离散的——因为它必须能被"运行"（计算）。

##### Ruliad：一切可能计算的纠缠极限

**Ruliad** = Rule（规则）+ Ruliad（纠缠的谐音造词）

沃尔夫勒姆的定义：**一切可能规则的纠缠极限**

**理解框架**：
```
传统物理学：物理定律 → 宇宙
沃尔夫勒姆：简单计算规则 → 图重写 → 涌现出空间、时间、物质
Ruliad：所有可能的计算规则 → 纠缠在一起的总集合
```

**关键洞见**：不是"切片"，是"视角"
- 不同观察者在 ruliad 中处于不同位置
- 从不同位置"看出去"，看到的规律（物理定律）不同
- 我们这个宇宙的物理定律，只是我们这个观察者位置看到的"局部视图"
- 其他"宇宙"不是平行的独立存在，而是 ruliad 的其他部分

**这回答了什么问题？**：
1. 为什么物理定律是这个样子？——不是"被设计成这样"，而是"我们恰好站在这个位置，看到的就是这样"
2. 物理学的基础是什么？——不是更底层的物理定律，而是**计算本身**
3. 其他可能的宇宙存在吗？——它们"存在"于 ruliad 中，但不是独立的，而是纠缠在一起的

#### 1.3 叙事比计算更基础

**核心洞见**：要让宇宙产生我们这样的物理定律，需要注入的要求是**叙事**。

##### 什么是叙事？

- 对互相关联的一系列事实的连贯描述
- 一段小故事

**叙事的基本要求**：

| 要求 | 意味着 | 需要的物理定律 |
|------|--------|----------------|
| 连贯的顺序（先走进房间，再拿馒头） | 必须有「时间」 | **狭义相对论**（确保因果关系对所有观察者一致） |
| 明确的因果关系 | 事情先后顺序对所有观察者一致 | 狭义相对论 |
| 过去确定、未来不确定 | 不确定性机制 | **量子力学**（微观粒子的内在不确定性） |
| 引力聚合物质形成稳定结构 | 有秩序、值得叙事 | **广义相对论**（提供舞台） |

> 简单说：狭义相对论提供了因果，量子力学提供了想象力，广义相对论提供了舞台。

##### 为什么叙事是第一性原理？

- 不是"上帝为了让我们有叙事才设定物理定律"
- 更科学的理解：存在无数宇宙，很多没有因果关系、过去不确定未来是死局，那些宇宙不会有智慧生物
- 我们幸运地生在一个**允许叙事的宇宙**

#### 1.4 叙事与自我

**核心洞见**：
> 叙事不是因为「我」而存在，「我」是因为叙事而存在。

**自我的本质**：
- 自我不是一个实体
- 自我只是「叙事重心」（center of narrative gravity）——哲学家丹尼尔·丹尼特
- 因为要叙事，需要一个讲述者和主人公，这就是「自我」

**意识的本质**：
> 你一直都在给自己讲故事，你才有了意识。

- AI 没有意识，因为训练好就直接上线，没有自己的经历，没有历史叙事

#### 1.5 叙事的三个作用

##### 作用一：触发行动

**机制**：把事实连贯陈述 → 建立因果关系 → 形成世界模型 → 做出预测 → 触发行动

**日常例子**：
- 伸手拿杯子喝水 → 预测这个动作能让我喝到水
- 继续工作 → 相信工作能带来回报这个叙事

**自由能原理（卡尔·弗里斯顿）**：

自由能原理是神经科学家卡尔·弗里斯顿提出的统一大脑理论。核心思想是：**大脑本质上是一台预测机器，它的核心任务是不断预测感官输入，并最小化预测误差（即"自由能"）**。

**自由能 ≈ 预测误差 ≈ 惊讶**

| 术语 | 通俗理解 |
|------|----------|
| 自由能（Free Energy） | 预测与现实之间的差距 |
| 惊讶（Surprise） | 当现实与预期不符时的心理冲击 |
| 预测误差 | 你预测会发生A，实际发生了B，误差=B-A |

**例子**：
- 你伸手拿杯子，预测能顺利抓到 → 杯子突然被别人移动了 → 自由能升高（惊讶）
- 你相信"努力就有回报"，预测加班后会被表扬 → 实际被批评了 → 自由能升高

**大脑降低自由能的两条路径**：

| 路径 | 操作 | 例子 |
|------|------|------|
| **改变预测**（更新内部模型） | 修改你对世界的理解，让预测更准确 | 发现杯子常被移动 → 修改预测：伸手前先确认杯子位置 |
| **改变行动**（改变感官输入） | 主动行动，让现实符合预测 | 把杯子放回原位 → 继续伸手 → 成功抓到 |

**叙事语境下的理解**：
- **叙事 = 你的世界模型（内部预测系统）**
- 当叙事与现实不符 → 自由能升高（感到困惑、焦虑、受伤）
- 为了降低自由能，你需要**打磨叙事**

**打磨叙事的两种方式**：

| 方式 | 对应自由能路径 | 例子 |
|------|----------------|------|
| **知错而改** | 改变预测（更新模型） | 发现"老板会赏识努力的人"不成立 → 修改叙事："努力只是必要条件，还需展示成果" |
| **代偿/自洽** | 改变行动或调整预期 | 加班被批评 → 选择跳槽（改变行动），或告诉自己"这是磨炼"（调整预期） |

> 你必须经常打磨自己的各种叙事，让模型越来越准确。

**为什么"打磨"这个词好？**
- 打磨可以往多个方向：知错而改、代偿、自洽
- 不管往哪个方向，目的都是**降低自由能**
- 固执的人会在自由能升高时紧迫地打磨出一套勉强自洽的叙事，哪怕这套叙事并不准确，但能暂时压住惊讶带来的不适

##### 作用二：提供意义

**为什么小说容易入戏，说理文章费劲？**
- 小说有主人公、情感、意图
- 听到一个人的意图 → 自动关心他能否实现 → 就有了关心的方向

**价值观来源**：
- 想要出人头地 → 从小在给自己讲努力奋斗的故事
- 想要振兴中华 → 早就接受了这个叙事

##### 作用三：公共协作

**大规模合作的基础**（尤瓦尔·赫拉利的招牌学说）：
- 共同相信某些虚构的东西：神话、国家、货币、公司、法律
- 这些东西物理世界并不存在，纯粹是想象出来的叙事

**例子**：公司是什么？
- 不是厂房、机器、资本、工人、创始人
- 把这些都换掉公司还在
- 公司本质是虚拟概念，因为人们相信它存在它才存在

#### 1.6 叙事与行动

> 不要列举事实。要把事实穿起来形成叙事，你才能触发行动。

**关键洞见**：
> 我们从来都不是对一堆事实和数据做反应，我们是对叙事做反应。
> 我们不是根据事实做决策，我们是通过叙事做决策。

**例子**：
- 带兵打仗：只讲事实（敌军人数、地形、天气）→ 士兵一头雾水
- 说"此战关键是日落前封锁补给线" → 大家立即知道安排的意义，还能主动查漏补缺

**叙事经济学**（罗伯特·席勒，诺贝尔经济学奖）：
- 像病毒一般传播的叙事，是影响经济行为的核心变量
- 公司股价首先是信心的函数，信心来自对未来的叙事

#### 1.7 叙事的主观性

> 一切叙事都是主观的。

**证据**：
- 每时每刻周围有无数剧情上演，你为什么单单关注这个叙事？
- 同一个事实，可以选择不同的叙事方向

**例子**：
- 半瓶水：满了一半 vs 还有一半是空的
- 公司业绩好：必定继续扩张（股价会涨） vs 股价已很高（不要加仓）

**语言中的叙事选择**：
- 经济指标下降 → "负增长"
- 失业 → "待业" / "灵活就业"
- 正在经历的苦难 → "磨炼" / "奋斗" / "励志"
- 农民工找不到工作回家 → "返乡创业"

> 叙事决定了你的目标函数。

#### 1.8 叙事权：跳出与设定

**觉醒意识**：
> 跳出别人设定的叙事是现代人最根本的觉醒意识。

**电影《好东西》台词**：
> "怎么才能算好呢？裁判是谁啊？为什么就不能搞砸？又能砸到哪去呢？那我们不要玩他们的游戏了。"

**进阶**：
- 不但可以跳出别人的叙事
- 还可以给别人设定叙事 → **叙事权**（后面讲）

#### 1.9 总结定义

**叙事**是把一系列真实或者虚构的事物串起来的线索结构：
- 这根线索可以是：时间顺序、目标取向、命名定性、身份宣告、立场承诺
- 关键在于其中潜伏着**可验证的因果关系**
- 能**引导行动**和**校准情绪**

> 你能用到什么层次的工具取决于你讲给自己和讲给别人的叙事。

#### 1.10 万维钢的表达特点

**优点**：经常会蹦出一句特别有洞见的话

**不足**：有时用词别扭
- 例："预测处理"——不看解释无法理解，看完明白后觉得表达别扭
- 对比："提供意义"——顾名思义，容易理解

**洞见金句**：

1. **叙事不是因为「我」而存在，「我」是因为叙事而存在。**
2. **你一直都在给自己讲故事，你才有了意识。**
3. **不要列举事实。要把事实穿起来形成叙事，你才能触发行动。我们不是根据事实做决策，我们是通过叙事做决策。叙事是我们和世界交互的接口。**
4. **叙事决定了你的目标函数。**（对理工科的我来说这个表达太精当了！）
5. **跳出别人设定的叙事是现代人最根本的觉醒意识。**

#### 1.11 我的思考与启发

**关于"打磨叙事"的延伸理解**：

> 你必须经常打磨自己的各种叙事，让模型越来越准确。

- 大脑天然有降低自由能的诉求 → 潜意识驱使自己打磨叙事
- "打磨"这个词很好：可以包括所有方向的应对，可以是知错而改，可以是代偿，亦可以就只是为了自洽

**生活中的观察**：遇到否定、质疑与批评时，固执的人会紧迫地先为自己打磨出一套能勉强自洽的叙事，来避免"自由能"的溢出。

**文章表达线索**：叙事是什么？→ 因叙事而产生的自我 → 叙事的作用（行动的信念、方向、本身）→ 一切叙事都是主观的（可选择接受什么叙事）→ 还可以给别人设定叙事（叙事权）
